## Section 4: SQL Writing Questions

**Tables:** employees(id, name, department_id, salary, hire_date), departments(id, dept_name, location), orders(id, customer_id, amount, order_date), customers(id, name, email, city)

**Q1.** Select all employees ordered by salary descending.
**Q2.** Find employees hired after 2022-01-01 with salary above 50000.
**Q3.** Count employees in each department.
**Q4.** Find the department with the highest average salary.
**Q5.** List employees with their department names using JOIN.
**Q6.** Find departments with more than 5 employees.
**Q7.** Get the top 3 highest-paid employees.
**Q8.** Find employees who earn more than the company average (subquery).
**Q9.** List customers who have never placed an order (LEFT JOIN).
**Q10.** Calculate total order amount per customer.
**Q11.** Find the customer with the most orders (subquery with COUNT).
**Q12.** Create a view showing employee name, department, and salary.
**Q13.** Find departments where no employee earns below 40000.
**Q14.** Get monthly order totals for 2023 using GROUP BY and date functions.
**Q15.** List employees and their rank by salary within each department (window function or subquery).

## Section 5: Answer Key

```sql
-- Q1
SELECT * FROM employees ORDER BY salary DESC;

-- Q2
SELECT * FROM employees WHERE hire_date > '2022-01-01' AND salary > 50000;

-- Q3
SELECT department_id, COUNT(*) AS emp_count FROM employees GROUP BY department_id;


-- Q4
SELECT department_id, AVG(salary) AS avg_sal FROM employees GROUP BY department_id ORDER BY avg_sal DESC LIMIT 1;

-- Q5
SELECT e.name, d.dept_name FROM employees e JOIN departments d ON e.department_id = d.id;

-- Q6
SELECT department_id, COUNT(*) AS cnt FROM employees GROUP BY department_id HAVING COUNT(*) > 5;

-- Q7
SELECT name, salary FROM employees ORDER BY salary DESC LIMIT 3;

-- Q8
SELECT name, salary FROM employees WHERE salary > (SELECT AVG(salary) FROM employees);

-- Q9
SELECT c.name FROM customers c LEFT JOIN orders o ON c.id = o.customer_id WHERE o.id IS NULL;

-- Q10
SELECT customer_id, SUM(amount) AS total FROM orders GROUP BY customer_id;

-- Q11
SELECT customer_id, COUNT(*) AS order_count FROM orders GROUP BY customer_id ORDER BY order_count DESC LIMIT 1;

-- Q12
CREATE VIEW emp_overview AS SELECT e.name, d.dept_name, e.salary FROM employees e JOIN departments d ON e.department_id = d.id;

-- Q13
SELECT department_id FROM employees GROUP BY department_id HAVING MIN(salary) >= 40000;

-- Q14
SELECT EXTRACT(MONTH FROM order_date) AS month, SUM(amount) AS total FROM orders WHERE EXTRACT(YEAR FROM order_date) = 2023 GROUP BY month ORDER BY month;

-- Q15
SELECT name, department_id, salary, RANK() OVER (PARTITION BY department_id ORDER BY salary DESC) AS rank FROM employees;
```
