# Day 173: Career Paths in Database Work

## Section 1: Topic Overview
Today explores four major career paths: Database Administrator (DBA), Data Engineer, Backend Developer, and Data Analyst. Understanding these roles helps you choose your specialization.

## Section 2: Simple English Explanation
Database skills open doors to many careers. A DBA manages servers. A Data Engineer builds pipelines and warehouses. A Backend Developer writes application code interacting with databases. A Data Analyst queries databases to find business insights. Each requires different skills but all need strong SQL.

## Section 4: Role Breakdown

### DBA (Database Administrator) - $70K-$130K
**Does:** Install, configure, secure, backup, optimize database servers
**Key skills:** PostgreSQL/MySQL/Oracle, Linux admin, backup/recovery, performance tuning, security
**Daily:** Monitor health, manage permissions, optimize queries, handle upgrades

### Data Engineer - $90K-$160K
**Does:** Build data infrastructure, pipelines, and warehouses
**Key skills:** SQL + Python, Spark/Airflow/dbt, Cloud (AWS/GCP/Azure), data modeling
**Daily:** Design ETL pipelines, maintain warehouses, ensure data quality

### Backend Developer - $80K-$150K
**Does:** Build application logic that reads/writes databases
**Key skills:** SQL + programming (Python/Java/Node.js), ORM frameworks, API design, migrations
**Daily:** Write queries, design schemas, optimize app queries, handle errors

### Data Analyst - $55K-$100K
**Does:** Query databases for business insights and reports
**Key skills:** Advanced SELECT queries, BI tools (Tableau/Power BI), statistics, communication
**Daily:** Write report queries, build dashboards, present findings, clean data

## Section 6: Technical Vocabulary
| Term | Definition |
|------|-----------|
| ETL | Extract, Transform, Load - data pipeline pattern |
| ORM | Object-Relational Mapping - code-to-database bridge |
| Data warehouse | Central repository for structured reporting data |
| Migration | Script that changes database schema version |
| Dashboard | Visual display of key metrics and data |
| CI/CD | Continuous Integration/Delivery - automated deployment |
