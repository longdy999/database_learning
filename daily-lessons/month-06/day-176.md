## Section 4: Exam Questions

**Q1.** What does SQL stand for?
a) Simple Query Logic  b) Structured Query Language  c) System Query Language  d) Standard Query Logic

**Q2.** Which key uniquely identifies each row in a table?
a) Foreign Key  b) Candidate Key  c) Primary Key  d) Index Key

**Q3.** Which SQL clause filters rows BEFORE grouping?
a) HAVING  b) WHERE  c) GROUP BY  d) ORDER BY

**Q4.** What does JOIN do in SQL?
a) Deletes rows  b) Combines rows from two or more tables  c) Creates indexes  d) Backs up data

**Q5.** Which normal form eliminates partial dependencies?
a) 1NF  b) 2NF  c) 3NF  d) BCNF

**Q6.** What does HAVING clause filter?
a) Individual rows  b) Grouped results  c) Table names  d) Column types

**Q7.** Which JOIN returns all rows from the left table?
a) INNER JOIN  b) RIGHT JOIN  c) LEFT JOIN  d) CROSS JOIN

**Q8.** What is an index used for?
a) Data encryption  b) Faster data retrieval  c) Table creation  d) User authentication

**Q9.** Which command removes all rows but keeps the table structure?
a) DROP  b) DELETE  c) TRUNCATE  d) REMOVE


**Q10.** What does a VIEW provide?
a) Physical table copy  b) Virtual table from a query  c) Backup file  d) Index structure

**Q11.** Which constraint prevents NULL values?
a) UNIQUE  b) CHECK  c) NOT NULL  d) DEFAULT

**Q12.** What is normalization?
a) Adding indexes  b) Reducing data redundancy  c) Creating backups  d) Encrypting data

**Q13.** Which aggregate function counts non-NULL values?
a) SUM()  b) COUNT(*)  c) COUNT(column)  d) AVG()

**Q14.** What does a foreign key represent?
a) Primary identifier  b) Link between two tables  c) Unique constraint  d) Index reference

**Q15.** Which transaction property ensures all-or-nothing execution?
a) Consistency  b) Isolation  c) Durability  d) Atomicity

**Q16.** What does EXPLAIN show in PostgreSQL?
a) Table structure  b) Query execution plan  c) User permissions  d) Database size

**Q17.** Which command creates a new database?
a) NEW DATABASE  b) CREATE DATABASE  c) MAKE DATABASE  d) ADD DATABASE

**Q18.** What is a subquery?
a) A deleted query  b) A query inside another query  c) A backup query  d) A view

**Q19.** Which clause limits the number of rows returned?
a) WHERE  b) LIMIT  c) HAVING  d) TOP

**Q20.** What does CASCADE do on DELETE?
a) Ignores deletion  b) Deletes related rows in child tables  c) Creates backup  d) Locks table

**Q21.** Which data type stores variable-length text?
a) CHAR  b) VARCHAR  c) INT  d) BOOLEAN

**Q22.** What is an ERD?
a) Error Recovery Document  b) Entity Relationship Diagram  c) External Resource Database  d) Entry Record Data

**Q23.** Which SQL operation combines results of two queries?
a) JOIN  b) UNION  c) MERGE  d) CONNECT

**Q24.** What does GRANT do?
a) Removes permissions  b) Creates tables  c) Gives permissions to users  d) Deletes users

**Q25.** Which backup type copies only changes since last backup?
a) Full backup  b) Incremental backup  c) Snapshot  d) Mirror backup
