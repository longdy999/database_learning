# Day 154: Exam Review — Aggregation and GROUP BY

## 1. Topic
Review of COUNT, SUM, AVG, MIN, MAX, GROUP BY, and HAVING clauses.

## 2. English Explanation
Aggregate functions perform calculations across multiple rows and return a single value. COUNT counts rows. SUM adds numeric values. AVG calculates the average. MIN/MAX find the smallest/largest value. GROUP BY groups rows that share a value so aggregates calculate per group. HAVING filters groups after aggregation (unlike WHERE which filters before). The execution order is: FROM → WHERE → GROUP BY → HAVING → SELECT → ORDER BY → LIMIT.

## 4. SQL Example

```sql
-- COUNT: Total number of orders
SELECT COUNT(*) AS total_orders FROM orders;

-- SUM: Total revenue
SELECT SUM(total_amount) AS total_revenue FROM orders;

-- AVG: Average order value
SELECT AVG(total_amount) AS avg_order_value FROM orders;

-- MIN/MAX: Price range
SELECT MIN(price) AS cheapest, MAX(price) AS most_expensive FROM products;

-- GROUP BY: Orders per customer
SELECT customer_id, COUNT(*) AS order_count, SUM(total_amount) AS total_spent
FROM orders
GROUP BY customer_id
ORDER BY total_spent DESC;

-- HAVING: Customers with more than 5 orders
SELECT customer_id, COUNT(*) AS order_count
FROM orders
GROUP BY customer_id
HAVING COUNT(*) > 5
ORDER BY order_count DESC;

-- Combined: Monthly revenue report
SELECT
    EXTRACT(MONTH FROM order_date) AS month,
    COUNT(*) AS num_orders,
    SUM(total_amount) AS revenue,
    AVG(total_amount) AS avg_order
FROM orders
WHERE order_date >= '2024-01-01'
GROUP BY EXTRACT(MONTH FROM order_date)
HAVING SUM(total_amount) > 1000
ORDER BY month;
```

## 6. Technical Vocabulary

| Term | Meaning |
|------|---------|
| COUNT(*) | Counts all rows including NULLs |
| COUNT(column) | Counts non-NULL values in that column |
| SUM | Adds all numeric values together |
| AVG | Calculates arithmetic mean |
| MIN/MAX | Finds smallest/largest value |
| GROUP BY | Groups rows with same values for aggregation |
| HAVING | Filters groups after aggregation |
| Aggregate Function | Function that operates on multiple rows |
| Alias (AS) | Gives a temporary name to a column or result |
| Execution Order | FROM → WHERE → GROUP BY → HAVING → SELECT → ORDER BY |
