# Day 51: Week 7 Review and Checkpoint

## Topic
Review and consolidate all aggregate functions and grouping concepts: COUNT, SUM, AVG, MIN, MAX, GROUP BY, and HAVING.

## English Explanation
Today is a checkpoint day. You will review everything you learned in Week 7 about aggregate functions. The goal is to write queries from memory without looking at notes. Aggregate functions summarize data across rows вҖ” COUNT counts rows, SUM adds values, AVG calculates averages, MIN finds the smallest value, MAX finds the largest value. GROUP BY organizes rows into groups before applying aggregates. HAVING filters groups after aggregation. You should be able to write all of these confidently before moving forward.

## SQL Example
```sql
-- Review: All aggregate functions in one query
SELECT
    department,
    COUNT(*) AS total_employees,
    SUM(salary) AS total_salary,
    AVG(salary) AS average_salary,
    MIN(salary) AS lowest_salary,
    MAX(salary) AS highest_salary
FROM employees
GROUP BY department
HAVING COUNT(*) >= 3
ORDER BY total_salary DESC;
```
