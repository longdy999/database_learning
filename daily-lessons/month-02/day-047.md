# Day 47: MIN and MAX - Finding Extremes

## Topic
Understanding MIN(column) and MAX(column) aggregate functions. MIN finds the smallest value. MAX finds the largest value. Both work with numbers, text (alphabetical order), and dates. Both skip NULL values.

## English Explanation
MIN and MAX are aggregate functions that find extreme values:

- **MIN(column)** returns the smallest value in a column
- **MAX(column)** returns the largest value in a column

They work with different data types:
- Numbers: smallest/largest number (e.g., cheapest price, highest salary)
- Text: first/last in alphabetical order (e.g., 'Apple' < 'Banana')
- Dates: earliest/latest date (e.g., first hire date, most recent order)

Like other aggregate functions, MIN and MAX skip NULL values entirely.

## SQL Example

### Setup Data
```sql
CREATE TABLE employees (
    emp_id INTEGER PRIMARY KEY,
    name TEXT NOT NULL,
    department TEXT,
    salary REAL,
    hire_date TEXT
);

INSERT INTO employees VALUES (1, 'Sokha', 'IT', 1500, '2020-03-15');
INSERT INTO employees VALUES (2, 'Dara', 'IT', 1200, '2021-06-01');
INSERT INTO employees VALUES (3, 'Chenda', 'HR', 1800, '2019-01-10');
INSERT INTO employees VALUES (4, 'Bopha', 'HR', NULL, '2022-09-20');
INSERT INTO employees VALUES (5, 'Vanna', 'Sales', 900, NULL);
INSERT INTO employees VALUES (6, 'Rith', 'Sales', 1100, '2023-02-14');
```


### MIN and MAX with Numbers
```sql
-- Lowest salary
SELECT MIN(salary) AS lowest_salary FROM employees;
-- Result: 900

-- Highest salary
SELECT MAX(salary) AS highest_salary FROM employees;
-- Result: 1800

-- Salary range
SELECT
    MIN(salary) AS lowest,
    MAX(salary) AS highest,
    MAX(salary) - MIN(salary) AS salary_range
FROM employees;
-- Result: lowest=900, highest=1800, salary_range=900
```

### MIN and MAX with Dates
```sql
-- Earliest hire date
SELECT MIN(hire_date) AS first_hired FROM employees;
-- Result: 2019-01-10

-- Most recent hire date
SELECT MAX(hire_date) AS last_hired FROM employees;
-- Result: 2023-02-14
```

### MIN and MAX with Text
```sql
-- Alphabetically first and last name
SELECT MIN(name) AS first_alpha, MAX(name) AS last_alpha
FROM employees;
-- Result: first_alpha='Bopha', last_alpha='Vanna'
```

### MIN and MAX with GROUP BY
```sql
-- Highest and lowest salary per department
SELECT
    department,
    MIN(salary) AS min_salary,
    MAX(salary) AS max_salary
FROM employees
WHERE department IS NOT NULL
GROUP BY department;
```
| department | min_salary | max_salary |
|------------|-----------|-----------|
| HR         | 1800      | 1800      |
| IT         | 1200      | 1500      |
| Sales      | 900       | 1100      |
