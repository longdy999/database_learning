## SQL Example

### Review: Multi-Table JOIN with Aggregation
```sql
-- Combine joins, grouping, and filtering in one query
SELECT
    c.customer_name,
    COUNT(o.order_id) AS total_orders,
    SUM(o.total_amount) AS total_spent,
    MAX(o.order_date) AS last_order_date
FROM customers c
LEFT JOIN orders o ON c.customer_id = o.customer_id
GROUP BY c.customer_name
HAVING COUNT(o.order_id) > 0
ORDER BY total_spent DESC;
```


### Review: Debugging a Complex Query
```sql
-- Find and fix all errors in this query:
-- BROKEN VERSION:
SELECT name, department, AVG(salary) AS avg_sal
FROM employees
JOIN departments
WHERE avg_sal > 50000
GROUP BY department;

-- FIXED VERSION:
SELECT e.name, d.department_name, AVG(e.salary) AS avg_sal
FROM employees e
JOIN departments d ON e.dept_id = d.dept_id
GROUP BY e.name, d.department_name
HAVING AVG(e.salary) > 50000;
```

### Review: Business Pattern - Top N with Percentage
```sql
-- Top 5 categories with their percentage of total sales
SELECT
    category,
    SUM(amount) AS category_total,
    ROUND(SUM(amount) * 100.0 / (SELECT SUM(amount) FROM sales), 2) AS pct
FROM sales
GROUP BY category
ORDER BY category_total DESC
LIMIT 5;
```
