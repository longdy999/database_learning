# Day 56: Common Query Patterns for Real Business Questions

## Topic
Common query patterns that solve real business questions: top N items, year-over-year comparison, running totals, percentage of total, and finding duplicates.

## English Explanation
Every business has recurring questions: "Who are our top 10 customers?" "How do this year's sales compare to last year?" "What percentage of revenue comes from each category?" "Are there duplicate records?" These questions map to specific SQL patterns. Once you learn these patterns, you can answer almost any business question quickly. Today we learn the 5 most important query patterns used in real companies every day.

## SQL Example

### Pattern 1: Top N Items
```sql
-- Business question: "Who are our top 5 customers by total spending?"
SELECT c.customer_name, SUM(o.total_amount) AS total_spent
FROM customers c
JOIN orders o ON c.customer_id = o.customer_id
GROUP BY c.customer_name
ORDER BY total_spent DESC
LIMIT 5;
```

### Pattern 2: Year-over-Year Comparison
```sql
-- Business question: "How do this year's monthly sales compare to last year?"
SELECT
    EXTRACT(MONTH FROM order_date) AS month,
    SUM(CASE WHEN EXTRACT(YEAR FROM order_date) = 2023 THEN total_amount ELSE 0 END) AS sales_2023,
    SUM(CASE WHEN EXTRACT(YEAR FROM order_date) = 2024 THEN total_amount ELSE 0 END) AS sales_2024,
    ROUND(
        (SUM(CASE WHEN EXTRACT(YEAR FROM order_date) = 2024 THEN total_amount ELSE 0 END) -
         SUM(CASE WHEN EXTRACT(YEAR FROM order_date) = 2023 THEN total_amount ELSE 0 END)) /
        NULLIF(SUM(CASE WHEN EXTRACT(YEAR FROM order_date) = 2023 THEN total_amount ELSE 0 END), 0) * 100,
    2) AS growth_percent
FROM orders
WHERE EXTRACT(YEAR FROM order_date) IN (2023, 2024)
GROUP BY EXTRACT(MONTH FROM order_date)
ORDER BY month;
```


### Pattern 3: Running Total
```sql
-- Business question: "What is the cumulative revenue each day this month?"
SELECT
    order_date,
    daily_total,
    SUM(daily_total) OVER (ORDER BY order_date) AS running_total
FROM (
    SELECT order_date, SUM(total_amount) AS daily_total
    FROM orders
    WHERE order_date >= '2024-01-01' AND order_date < '2024-02-01'
    GROUP BY order_date
) daily_sales
ORDER BY order_date;
```

### Pattern 4: Percentage of Total
```sql
-- Business question: "What percentage of sales comes from each category?"
SELECT
    p.category,
    SUM(oi.quantity * oi.unit_price) AS category_sales,
    ROUND(
        SUM(oi.quantity * oi.unit_price) * 100.0 /
        (SELECT SUM(quantity * unit_price) FROM order_items),
    2) AS percent_of_total
FROM order_items oi
JOIN products p ON oi.product_id = p.product_id
GROUP BY p.category
ORDER BY percent_of_total DESC;
```

### Pattern 5: Finding Duplicates
```sql
-- Business question: "Are there duplicate customer email addresses?"
SELECT email, COUNT(*) AS occurrence_count
FROM customers
GROUP BY email
HAVING COUNT(*) > 1
ORDER BY occurrence_count DESC;
```
