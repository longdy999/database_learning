Comprehensive review of Week 5 concepts: ORDER BY, LIMIT, OFFSET, LIKE, and DISTINCT.

## English Explanation

This review ensures no gaps in your foundation. If you cannot write these queries from memory, spend extra time practicing before moving forward.

## SQL Example

```sql
-- ORDER BY review
SELECT * FROM products ORDER BY price DESC;
SELECT * FROM employees ORDER BY department, last_name ASC;

-- LIMIT and OFFSET review
SELECT * FROM products ORDER BY price DESC LIMIT 5;
SELECT * FROM products ORDER BY price DESC LIMIT 10 OFFSET 20;

-- LIKE review
SELECT * FROM customers WHERE name LIKE 'S%';
SELECT * FROM products WHERE description LIKE '%organic%';
SELECT * FROM employees WHERE email LIKE '%@gmail.com';

-- DISTINCT review
SELECT DISTINCT category FROM products;
SELECT COUNT(DISTINCT city) FROM customers;

-- Combined query using all concepts
SELECT DISTINCT category FROM products
WHERE name LIKE '%Pro%'
ORDER BY category ASC
LIMIT 5;
```

## Reading Practice

Review the following query and explain what each line does:

```sql
SELECT DISTINCT category, COUNT(*) as product_count
FROM products
WHERE name LIKE '%smart%'
GROUP BY category
ORDER BY product_count DESC
LIMIT 3 OFFSET 0;
```
