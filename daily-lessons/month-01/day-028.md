## SQL Example
```sql
-- Week 4 Review Exercises:

-- Exercise 1: UPDATE with conditions
UPDATE employees SET salary = salary * 1.10
WHERE department = 'Sales' AND hire_year < 2020;

-- Exercise 2: DELETE with safety check
SELECT * FROM orders WHERE status = 'cancelled' AND order_date < '2023-01-01';
DELETE FROM orders WHERE status = 'cancelled' AND order_date < '2023-01-01';

-- Exercise 3: Complex WHERE
SELECT * FROM products
WHERE (category IN ('Electronics', 'Software'))
AND (price BETWEEN 10 AND 100)
AND (stock IS NOT NULL);


-- Exercise 4: NULL handling
SELECT name, COALESCE(email, 'No email') AS contact
FROM customers WHERE phone IS NULL;
```

## Hands-on Practice
1. Write an UPDATE that gives a 10% raise to employees hired before 2020
2. Write a safe DELETE (SELECT first, then DELETE) for old cancelled orders
3. Write a query combining IN, BETWEEN, and IS NOT NULL
4. Fix this broken query: SELECT * FROM users WHERE email = NULL
5. Self-assessment: rate your confidence 1-5 on each topic this week
