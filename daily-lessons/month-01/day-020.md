# Day 20: WHERE with Numbers - >, <, >=, <=

## Topic
Numeric comparison operators in WHERE clause: greater than, less than, greater than or equal, less than or equal.

## English Explanation
Yesterday we learned = and !=. Today we add **numeric comparisons**:

| Operator | Meaning | Example |
|----------|---------|---------|
| > | Greater than | `WHERE price > 100` |
| < | Less than | `WHERE stock < 10` |
| >= | Greater than or equal | `WHERE age >= 18` |
| <= | Less than or equal | `WHERE salary <= 2000` |

**Key points:**
- These operators work with numbers (INTEGER, REAL)
- They also work with dates stored as text in YYYY-MM-DD format
- You can combine them with column selection and aliases
- No quotes needed around numbers: `WHERE price > 50` (not `'50'`)

**Common real-world uses:**
- Find expensive products: `WHERE price > 100`
- Find low stock items: `WHERE stock < 5`
- Find adults: `WHERE age >= 18`
- Find budget items: `WHERE price <= 20`

## Khmer Explanation (ការពន្យល់ជាភាសាខ្មែរ)
ការប្រើ WHERE ជាមួយលេខ — ធំជាង តូចជាង ស្មើ។
> មានន័យថាធំជាង។ < មានន័យថាតូចជាង។
>= មានន័យថាធំជាងឬស្មើ។ <= មានន័យថាតូចជាងឬស្មើ។
នេះប្រើសម្រាប់ត្រងទិន្នន័យដែលជាលេខ ដូចជា តម្លៃ អាយុ ប្រាក់ខែ។


## SQL Example
```sql
-- Setup: Products table with prices and stock
CREATE TABLE products (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT NOT NULL,
    category TEXT NOT NULL,
    price REAL NOT NULL,
    stock INTEGER NOT NULL,
    rating REAL DEFAULT 0.0
);

INSERT INTO products (name, category, price, stock, rating)
VALUES
    ('Basic Mouse', 'Electronics', 12.99, 150, 3.5),
    ('Gaming Mouse', 'Electronics', 79.99, 25, 4.7),
    ('4K Monitor', 'Electronics', 450.00, 8, 4.8),
    ('USB Cable', 'Accessories', 5.99, 300, 4.0),
    ('Laptop Stand', 'Furniture', 45.00, 40, 4.3),
    ('Ergonomic Chair', 'Furniture', 350.00, 5, 4.6),
    ('Notebook', 'Stationery', 3.50, 500, 4.1),
    ('Pen Pack', 'Stationery', 8.99, 200, 3.8),
    ('Webcam HD', 'Electronics', 89.99, 15, 4.4),
    ('Standing Desk', 'Furniture', 599.99, 3, 4.9);

-- Greater than (>)
SELECT name, price FROM products WHERE price > 100;
-- Returns: 4K Monitor, Ergonomic Chair, Standing Desk

-- Less than (<)
SELECT name, stock FROM products WHERE stock < 10;
-- Returns: 4K Monitor (8), Ergonomic Chair (5), Standing Desk (3)

-- Greater than or equal (>=)
SELECT name, price FROM products WHERE price >= 79.99;
-- Returns: Gaming Mouse, 4K Monitor, Ergonomic Chair, Webcam HD, Standing Desk

-- Less than or equal (<=)
SELECT name, price FROM products WHERE price <= 10;
-- Returns: USB Cable (5.99), Notebook (3.50), Pen Pack (8.99)

-- Combined with specific columns and aliases
SELECT name AS "Product", price AS "Price (USD)"
FROM products WHERE price > 50;

-- Using with rating
SELECT name, rating FROM products WHERE rating >= 4.5;
-- Returns: Gaming Mouse, 4K Monitor, Ergonomic Chair, Standing Desk
```


## Hands-on Practice (10-15 minutes)
Write queries filtering by price, salary, age, and stock:

```sql
-- Setup: Employees with salary data
CREATE TABLE staff (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT NOT NULL,
    age INTEGER NOT NULL,
    department TEXT NOT NULL,
    salary REAL NOT NULL,
    years_experience INTEGER NOT NULL DEFAULT 0
);

INSERT INTO staff (name, age, department, salary, years_experience)
VALUES
    ('Sokha', 28, 'IT', 1500, 4),
    ('Dara', 35, 'Marketing', 1200, 8),
    ('Bopha', 24, 'IT', 1800, 2),
    ('Vanna', 42, 'Sales', 2200, 15),
    ('Channa', 19, 'Intern', 400, 0),

    ('Pich', 26, 'IT', 1350, 3),
    ('Navy', 38, 'Finance', 2500, 12);

-- 1. Find employees earning more than $1500
SELECT name, salary FROM staff WHERE salary > 1500;

-- 2. Find employees younger than 30
SELECT name, age FROM staff WHERE age < 30;

-- 3. Find employees with 5 or more years experience
SELECT name, years_experience FROM staff WHERE years_experience >= 5;

-- 4. Find employees earning $1200 or less
SELECT name, salary, department FROM staff WHERE salary <= 1200;

-- 5. Find employees older than 35
SELECT name, age, department FROM staff WHERE age > 35;

-- 6. Find products with stock less than 20 (use products table)
SELECT name, stock FROM products WHERE stock < 20;

-- 7. Find products priced at $50 or more
SELECT name, price, category FROM products WHERE price >= 50;

-- 8. Find products with rating above 4.5
SELECT name, rating FROM products WHERE rating > 4.5;

-- 9. Find staff with less than 3 years experience
SELECT name, years_experience FROM staff WHERE years_experience < 3;

-- 10. Find products with stock of 100 or more
SELECT name, stock, price FROM products WHERE stock >= 100;
```

## Technical Vocabulary

| Term | Meaning | Khmer |
|------|---------|-------|
| Greater than (>) | More than a value | ធំជាង |
| Less than (<) | Fewer than a value | តូចជាង |
| Greater or equal (>=) | At least this value | ធំជាងឬស្មើ |
| Less or equal (<=) | At most this value | តូចជាងឬស្មើ |
| Comparison operator | Symbols that compare values | សញ្ញាប្រៀបធៀប |
| Numeric filter | Filtering based on number values | ការត្រងដោយលេខ |
