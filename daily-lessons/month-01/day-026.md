# Day 26: NULL Values - IS NULL, IS NOT NULL

## Topic
Understanding NULL values and how to filter for missing data.

## English Explanation
NULL means "no value" — it is not zero, not an empty string, not false. It means the data is unknown or missing. You cannot use = NULL because NULL is not equal to anything, not even itself. Instead, use IS NULL to find missing data and IS NOT NULL to find records that have data. Understanding NULL is critical for writing accurate queries.

## SQL Example
```sql
-- Create a table with some NULL values
CREATE TABLE customers (
    id INT PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    email VARCHAR(100),
    phone VARCHAR(20),
    city VARCHAR(50)
);

-- Insert data with some missing values
INSERT INTO customers VALUES (1, 'Dara', 'dara@email.com', '012345678', 'Phnom Penh');
INSERT INTO customers VALUES (2, 'Srey', NULL, '098765432', 'Siem Reap');
INSERT INTO customers VALUES (3, 'Vanny', 'vanny@email.com', NULL, NULL);
INSERT INTO customers VALUES (4, 'Kosal', NULL, NULL, 'Battambang');


-- WRONG: This does NOT work!
SELECT * FROM customers WHERE email = NULL;  -- Returns nothing!

-- CORRECT: Use IS NULL
SELECT * FROM customers WHERE email IS NULL;

-- Find customers who HAVE a phone number
SELECT * FROM customers WHERE phone IS NOT NULL;

-- Find customers missing city information
SELECT * FROM customers WHERE city IS NULL;

-- Combine with other conditions
SELECT * FROM customers
WHERE email IS NULL AND city IS NOT NULL;
```

## Speaking Practice
Explain to a client:
"NULL means the data is missing or unknown. It is different from zero or empty. For example, if a customer has no phone number in our system, that field is NULL. We use special commands IS NULL and IS NOT NULL to find these records. You cannot search for NULL the normal way — it requires its own syntax."

## Reading Practice
Read this query and predict the result:
```sql
SELECT name, email, phone FROM customers
WHERE email IS NULL OR phone IS NULL;
```
Which customers appear? Why? (Answer: Srey, Vanny, and Kosal — each is missing email or phone or both.)
