## SQL Example
```sql

-- 1. Create a table
CREATE TABLE movies (
    id INTEGER PRIMARY KEY,
    title TEXT NOT NULL,
    genre TEXT,
    year INTEGER,
    rating REAL
);

-- 2. Insert data
INSERT INTO movies VALUES (1, 'Parasite', 'Thriller', 2019, 8.5);
INSERT INTO movies VALUES (2, 'Inception', 'Sci-Fi', 2010, 8.8);
INSERT INTO movies VALUES (3, 'Frozen', 'Animation', 2013, 7.4);

-- 3. Read data - multiple ways
SELECT * FROM movies;
SELECT title, year FROM movies;
SELECT title AS movie_name, rating FROM movies;
```

## Speaking Practice
**Explain databases to an imaginary client (2 minutes):**

"Let me explain how we'll organize your business data. A database is like a digital filing system. We create tables — think of them as organized spreadsheets. Each table holds one type of information. For example, one table for customers, one for products, one for orders. I use a language called SQL to add data, find data, update data, or remove data. Today I'll show you a simple demonstration."
