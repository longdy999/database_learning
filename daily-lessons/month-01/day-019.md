# Day 19: WHERE Clause - Filtering with = and !=

## Topic
WHERE clause: filtering rows based on conditions using equality (=) and inequality (!=) operators.

## English Explanation
The **WHERE clause** filters which rows are returned by a query. Without WHERE, you get ALL rows. With WHERE, you get only rows that match your condition.

**Equality (=):** Returns rows where the column exactly matches a value
```
SELECT * FROM table WHERE column = 'value';
```

**Inequality (!= or <>):** Returns rows where the column does NOT match
```
SELECT * FROM table WHERE column != 'value';
SELECT * FROM table WHERE column <> 'value';
```

**Important rules:**
- Text values must be in single quotes: `WHERE name = 'Sokha'`
- Numbers do NOT need quotes: `WHERE age = 25`
- Comparison is case-sensitive in most databases
- NULL values are NOT matched by = or != (use IS NULL instead)
- WHERE goes AFTER FROM, BEFORE ORDER BY

## SQL Example
```sql
-- Setup table with varied data
CREATE TABLE employees (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT NOT NULL,
    department TEXT NOT NULL,
    position TEXT NOT NULL,
    salary REAL NOT NULL,
    city TEXT NOT NULL,
    status TEXT NOT NULL DEFAULT 'active'
);

INSERT INTO employees (name, department, position, salary, city, status)
VALUES
    ('Sokha Chem', 'IT', 'Developer', 1500, 'Phnom Penh', 'active'),
    ('Dara Phan', 'Marketing', 'Designer', 1200, 'Siem Reap', 'active'),
    ('Bopha Kim', 'IT', 'Senior Dev', 1800, 'Phnom Penh', 'active'),
    ('Vanna Sok', 'Sales', 'Manager', 1600, 'Battambang', 'inactive'),
    ('Channa Ly', 'HR', 'Officer', 1100, 'Phnom Penh', 'active'),
    ('Maly Heng', 'IT', 'Junior Dev', 900, 'Siem Reap', 'active'),
    ('Pich Oun', 'Sales', 'Rep', 1000, 'Phnom Penh', 'inactive'),
    ('Srey Mom', 'Marketing', 'Manager', 1700, 'Phnom Penh', 'active');

-- WHERE with = (equality)
SELECT * FROM employees WHERE department = 'IT';
-- Returns: Sokha, Bopha, Maly (3 rows)

SELECT * FROM employees WHERE city = 'Phnom Penh';
-- Returns: Sokha, Bopha, Channa, Pich, Srey Mom (5 rows)

SELECT * FROM employees WHERE status = 'active';
-- Returns: 6 rows (all except Vanna and Pich)

-- WHERE with != (inequality)
SELECT * FROM employees WHERE department != 'IT';
-- Returns: Dara, Vanna, Channa, Pich, Srey Mom (5 rows)

SELECT * FROM employees WHERE status != 'active';
-- Returns: Vanna, Pich (2 rows)

SELECT * FROM employees WHERE city != 'Phnom Penh';
-- Returns: Dara, Vanna, Maly (3 rows)

-- Combining with specific columns
SELECT name, salary FROM employees WHERE department = 'IT';
SELECT name, department FROM employees WHERE city != 'Siem Reap';
```
