## SQL Example
```sql
CREATE TABLE contacts (
    id INT PRIMARY KEY,
    first_name VARCHAR(50) NOT NULL,
    last_name VARCHAR(50) NOT NULL,
    phone VARCHAR(20),
    email VARCHAR(100),
    city VARCHAR(50),
    category VARCHAR(30) DEFAULT 'personal',
    notes TEXT,
    created_date DATE NOT NULL
);

SELECT first_name, last_name, phone FROM contacts WHERE city = 'Phnom Penh' AND category = 'work';
```

## Hands-on Practice
Build the entire contact book system in 30 minutes:
1. Design and CREATE the contacts table with appropriate data types
2. INSERT at least 8 contacts with varied data (some NULL phone, some NULL email)
3. Write 5 useful queries: search by name, filter by city, find missing emails (IS NULL), update a phone number, delete a duplicate contact
