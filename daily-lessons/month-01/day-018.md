# Day 18: SELECT Specific Columns - Choosing What Data to Show

## Topic
SELECT col1, col2 FROM table: choosing specific columns, controlling column order in output, and using aliases with AS.

## English Explanation
Instead of SELECT * (all columns), you can **select only the columns you need**:

```
SELECT column1, column2 FROM table_name;
```

**Key concepts today:**
1. **Column selection:** Only retrieve what you need — faster and cleaner
2. **Column order in output:** Columns appear in the order you list them
3. **Aliases with AS:** Rename columns in the output for clarity
   ```
   SELECT first_name AS "First Name" FROM employees;
   ```
4. **Expressions in SELECT:** You can do math or combine columns
   ```
   SELECT name, price * 0.9 AS discounted_price FROM products;
   ```

**Professional rule:** Always select only the columns your application needs. Never use SELECT * in production queries.

## Khmer Explanation (ការពន្យល់ជាភាសាខ្មែរ)
ជ្រើសរើសជួរឈរជាក់លាក់ដែលត្រូវបង្ហាញ។ ជំនួស SELECT * យើងអាចជ្រើសរើស
តែជួរឈរដែលយើងត្រូវការ។ នេះលឿនជាង និងស្អាតជាង។
យើងអាចប្តូរឈ្មោះជួរឈរក្នុងលទ្ធផលដោយប្រើ AS។

## SQL Example
```sql
-- Setup table
CREATE TABLE employees (
    emp_id INTEGER PRIMARY KEY AUTOINCREMENT,
    first_name TEXT NOT NULL,
    last_name TEXT NOT NULL,
    email TEXT NOT NULL,
    department TEXT NOT NULL,
    position TEXT NOT NULL,
    salary REAL NOT NULL,
    hire_date TEXT NOT NULL DEFAULT (date('now'))
);


INSERT INTO employees (first_name, last_name, email, department, position, salary)
VALUES
    ('Sokha', 'Chem', 'sokha@company.com', 'IT', 'Developer', 1500.00),
    ('Dara', 'Phan', 'dara@company.com', 'Marketing', 'Designer', 1200.00),
    ('Bopha', 'Kim', 'bopha@company.com', 'IT', 'Senior Dev', 1800.00),
    ('Vanna', 'Sok', 'vanna@company.com', 'Sales', 'Manager', 1600.00),
    ('Channa', 'Ly', 'channa@company.com', 'HR', 'Officer', 1100.00);

-- Select only names
SELECT first_name, last_name FROM employees;

-- Select name and salary only
SELECT first_name, salary FROM employees;

-- Change column order in output
SELECT salary, first_name, department FROM employees;

-- Use aliases to rename columns in output
SELECT first_name AS "First Name", last_name AS "Last Name" FROM employees;

-- Calculated column with alias
SELECT first_name, salary, salary * 12 AS annual_salary FROM employees;

-- Combine columns (concatenation depends on database)
SELECT first_name || ' ' || last_name AS full_name, department FROM employees;

-- Select with alias for clarity in reports
SELECT
    department AS "Department",
    position AS "Job Title",
    salary AS "Monthly Pay (USD)"
FROM employees;
```

## Hands-on Practice (10-15 minutes)
Write 10 different SELECT statements using specific columns:
```sql
-- Practice table
CREATE TABLE products (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT NOT NULL,
    brand TEXT NOT NULL,
    category TEXT NOT NULL,
    price REAL NOT NULL,
    cost REAL NOT NULL,
    stock INTEGER NOT NULL DEFAULT 0,
    rating REAL DEFAULT 0.0
);


INSERT INTO products (name, brand, category, price, cost, stock, rating)
VALUES
    ('Wireless Mouse', 'Logitech', 'Electronics', 29.99, 15.00, 50, 4.5),
    ('Mechanical Keyboard', 'Corsair', 'Electronics', 89.99, 45.00, 30, 4.7),
    ('USB Hub', 'Anker', 'Electronics', 24.99, 10.00, 100, 4.2),
    ('Desk Lamp', 'IKEA', 'Furniture', 35.00, 18.00, 25, 4.0),
    ('Notebook A5', 'Moleskine', 'Stationery', 15.99, 5.00, 200, 4.8);

-- 1. Select only name and price
SELECT name, price FROM products;

-- 2. Select brand and category
SELECT brand, category FROM products;

-- 3. Select with profit calculation
SELECT name, price - cost AS profit FROM products;

-- 4. Select with alias for report
SELECT name AS "Product", price AS "Retail Price" FROM products;

-- 5. Reverse column order
SELECT stock, price, name FROM products;

-- 6. Select with percentage markup
SELECT name, cost, price, ((price - cost) / cost * 100) AS markup_percent FROM products;

-- 7. Select only stock-related info
SELECT name, stock, stock * price AS inventory_value FROM products;

-- 8. Select for a simple display
SELECT name AS "Item", brand AS "Made By", rating AS "Stars" FROM products;

-- 9. Select with combined text
SELECT name || ' by ' || brand AS product_label, price FROM products;

-- 10. Select for finance report
SELECT name, cost AS "Unit Cost", price AS "Sell Price", price - cost AS "Margin" FROM products;
```

## Technical Vocabulary

| Term | Meaning | Khmer |
|------|---------|-------|
| Column selection | Choosing specific columns to retrieve | ការជ្រើសរើសជួរឈរ |
| Alias (AS) | Temporary rename for a column in output | ឈ្មោះបណ្តោះអាសន្ន |
| Expression | A calculation in the SELECT list | កន្សោមគណនា |
| Concatenation | Joining text values together | ការភ្ជាប់អត្ថបទ |
| Output order | The column sequence in results | លំដាប់ជួរឈរក្នុងលទ្ធផល |
| Projection | Database term for selecting columns | ការជ្រើសរើសជួរឈរ (ពាក្យបច្ចេកទេស) |

## Speaking Practice
"In professional work, I always select only the columns I need rather than using SELECT star. This makes queries faster and the output easier to understand. I also use aliases with the AS keyword to give columns clear, readable names — especially when creating reports for clients who may not understand technical column names like emp_id or dept_code."
