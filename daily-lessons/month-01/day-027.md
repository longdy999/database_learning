# Day 27: Combining Everything - CRUD Practice Day

## Topic
Practice all CRUD operations: CREATE TABLE, INSERT, SELECT with WHERE, UPDATE, DELETE. Build a complete small system from scratch in 30 minutes.

## English Explanation
CRUD stands for Create, Read, Update, Delete — the four basic database operations. Today is a practice day where you combine everything you have learned in Month 1. You will design a table, insert data, query it with various WHERE conditions, update records, and delete records. This simulates real work.

## SQL Example
```sql
-- STEP 1: CREATE TABLE
CREATE TABLE books (
    id INT PRIMARY KEY,
    title VARCHAR(200) NOT NULL,
    author VARCHAR(100) NOT NULL,
    price DECIMAL(8,2),
    genre VARCHAR(50),
    published_year INT,
    in_stock BOOLEAN DEFAULT TRUE
);


-- STEP 2: INSERT data
INSERT INTO books VALUES (1, 'SQL Basics', 'John Smith', 25.99, 'Technology', 2022, TRUE);
INSERT INTO books VALUES (2, 'Database Design', 'Jane Doe', 32.50, 'Technology', 2021, TRUE);
INSERT INTO books VALUES (3, 'Khmer History', 'Sok Dara', 15.00, 'History', 2020, FALSE);
INSERT INTO books VALUES (4, 'Cooking Guide', 'Mary Lee', 18.75, 'Lifestyle', 2023, TRUE);
INSERT INTO books VALUES (5, 'Data Science', 'Bob Chen', 45.00, 'Technology', 2023, TRUE);

-- STEP 3: SELECT with WHERE
SELECT * FROM books WHERE genre = 'Technology' AND price < 40;
SELECT title, author FROM books WHERE published_year BETWEEN 2021 AND 2023;
SELECT * FROM books WHERE genre IN ('Technology', 'History');
SELECT * FROM books WHERE price IS NOT NULL AND in_stock = TRUE;

-- STEP 4: UPDATE
UPDATE books SET price = 29.99 WHERE id = 1;
UPDATE books SET in_stock = TRUE WHERE id = 3;

-- STEP 5: DELETE
DELETE FROM books WHERE id = 3;
```

## Hands-on Practice

2. Insert at least 5 books with varied data
3. Write 5 different SELECT queries using WHERE, AND, OR, IN, BETWEEN
4. Update 2 records (change price, change stock status)
5. Delete 1 record and verify it is gone
