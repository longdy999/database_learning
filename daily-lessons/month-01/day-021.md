# Day 21: Week 3 Review and Checkpoint

## Topic
Week 3 Review and Checkpoint — consolidating INSERT, SELECT, and WHERE clause skills.

## English Explanation
Today is a review day. You will practice everything from Week 3: INSERT statements, basic SELECT queries, and WHERE clauses with comparison operators (=, !=, >, <, >=, <=). The goal is to write 15 queries from memory without looking at notes. This builds real confidence and prepares you for more complex topics next week.

## SQL Example
```sql
-- Review INSERT
INSERT INTO products (name, price, category)
VALUES ('Keyboard', 25.99, 'Electronics');

-- Review SELECT with WHERE operators
SELECT * FROM products WHERE price > 20;
SELECT name, price FROM products WHERE category = 'Electronics';
SELECT * FROM products WHERE price >= 10 AND price <= 50;
SELECT name FROM products WHERE category != 'Food';
SELECT * FROM products WHERE price < 100 ORDER BY price;
```

7. SELECT WHERE column > value
8. SELECT WHERE column < value
9. SELECT WHERE column >= value
10. SELECT WHERE column <= value
11. INSERT multiple rows
12. SELECT with ORDER BY ASC
13. SELECT with ORDER BY DESC
14. SELECT with WHERE and ORDER BY combined
15. SELECT with LIMIT
