# Day 4: Introduction to SQL

## Topic
Understanding what SQL is, why it matters, and how it works.

## English Explanation
**SQL** (Structured Query Language) is the language we use to talk to databases. Just like you use English to communicate with people, you use SQL to communicate with databases.

SQL can do four main things (CRUD):
- **C**reate data (INSERT)
- **R**ead data (SELECT)
- **U**pdate data (UPDATE)
- **D**elete data (DELETE)

SQL is NOT a programming language like Python or Java. It is a **query language** — you tell the database WHAT you want, not HOW to get it.

## SQL Example
```sql
-- CREATE: Add new data
INSERT INTO students (name, age) VALUES ('Sokha', 18);

-- READ: Get data
SELECT * FROM students;

-- UPDATE: Change data
UPDATE students SET age = 19 WHERE name = 'Sokha';

-- DELETE: Remove data
DELETE FROM students WHERE name = 'Sokha';
```

## Speaking Practice
**Say this out loud:**

"SQL is the standard language for working with databases. When we need to find specific data, we write a SELECT query. When we need to add new data, we use INSERT. To change existing data, we use UPDATE. And to remove data, we use DELETE. These four operations cover most of what we do with databases daily."

## Reading Practice
> "SQL was created in the 1970s and remains the standard language for relational databases. Almost every database system supports SQL — PostgreSQL, MySQL, SQLite, Oracle, and SQL Server all use SQL with minor differences."
