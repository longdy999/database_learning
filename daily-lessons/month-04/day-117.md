# Day 117: Stored Procedures with Control Flow

## Topic
Using control flow statements (IF/ELSE, LOOP, WHILE) and variables inside stored procedures for complex business logic.

## English Explanation
Stored procedures become powerful when you add control flow. You can use IF/ELSIF/ELSE to make decisions, LOOP and WHILE to repeat actions, DECLARE to create variables, and RAISE NOTICE to print debug messages. This lets you write complete business logic — like applying different discounts based on order amount or customer loyalty level.

## SQL Example

```sql
-- Procedure with IF/ELSE and variables
CREATE OR REPLACE FUNCTION apply_discount(
    p_order_id INT
)
RETURNS NUMERIC AS $$
DECLARE
    v_total NUMERIC;
    v_discount NUMERIC := 0;
    v_final_price NUMERIC;
BEGIN
    -- Get the order total
    SELECT total_amount INTO v_total
    FROM orders WHERE order_id = p_order_id;

    -- Apply discount based on amount
    IF v_total >= 500 THEN
        v_discount := 0.15;
        RAISE NOTICE 'Premium discount: 15%%';
    ELSIF v_total >= 200 THEN
        v_discount := 0.10;
        RAISE NOTICE 'Standard discount: 10%%';
    ELSIF v_total >= 100 THEN
        v_discount := 0.05;
        RAISE NOTICE 'Basic discount: 5%%';
    ELSE
        RAISE NOTICE 'No discount applied';
    END IF;


    v_final_price := v_total - (v_total * v_discount);

    -- Update the order with discounted price
    UPDATE orders SET final_amount = v_final_price
    WHERE order_id = p_order_id;

    RETURN v_final_price;
END;
$$ LANGUAGE plpgsql;

-- Example with WHILE loop
CREATE OR REPLACE FUNCTION generate_monthly_report(p_year INT)
RETURNS VOID AS $$
DECLARE
    v_month INT := 1;
BEGIN
    WHILE v_month <= 12 LOOP
        RAISE NOTICE 'Processing month: %', v_month;
        -- Insert summary for each month
        INSERT INTO monthly_summary (year, month, generated_at)
        VALUES (p_year, v_month, NOW());
        v_month := v_month + 1;
    END LOOP;
END;
$$ LANGUAGE plpgsql;
```

## Technical Vocabulary

| Term | Definition |
|------|-----------|
| IF/ELSIF/ELSE | Conditional branching in plpgsql |
| LOOP | Repeats a block until EXIT is called |
| WHILE | Repeats a block while condition is true |
| DECLARE | Section to define local variables |
| RAISE NOTICE | Prints a debug message to the console |
| := | Assignment operator in plpgsql |
| INTO | Captures a query result into a variable |
| END IF | Closes an IF block |
| END LOOP | Closes a LOOP or WHILE block |
| %% | Escaped percent sign in RAISE NOTICE |
