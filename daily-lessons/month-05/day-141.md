# Day 141: Week 19 Review - Database Security

## Topic
Comprehensive review of database security concepts: users, roles, permissions, SQL injection prevention, encryption, and audit logs.

## English Explanation
Database security protects your data from unauthorized access, modification, or destruction. This review covers four pillars: (1) Users and Roles manage WHO can access data, (2) SQL Injection prevention stops attackers from manipulating queries, (3) Encryption protects data at rest and in transit, and (4) Audit Logs track WHO did WHAT and WHEN. A complete security plan combines all four pillars together.

## SQL Example
```sql
-- 1. Create roles and assign permissions
CREATE ROLE app_readonly;
GRANT SELECT ON ALL TABLES IN SCHEMA public TO app_readonly;

CREATE ROLE app_writer;
GRANT SELECT, INSERT, UPDATE ON ALL TABLES IN SCHEMA public TO app_writer;

-- 2. Create user with specific role
CREATE USER report_user WITH PASSWORD 'secure_password_123';
GRANT app_readonly TO report_user;

-- 3. Prevent SQL injection: Use parameterized queries
-- BAD (vulnerable):  "SELECT * FROM users WHERE id = " + user_input
-- GOOD (safe):       "SELECT * FROM users WHERE id = $1" with parameter binding

-- 4. Audit log table
CREATE TABLE audit_log (
    id SERIAL PRIMARY KEY,
    user_name VARCHAR(100),
    action VARCHAR(50),
    table_name VARCHAR(100),
    record_id INTEGER,
    old_value JSONB,
    new_value JSONB,
    performed_at TIMESTAMP DEFAULT NOW()
);
```

## Technical Vocabulary

| English Term | Khmer Translation | Definition |
|---|---|---|
| Role | តួនាទី | A named set of database permissions |
| GRANT | ផ្តល់សិទ្ធិ | Give permissions to a user or role |
| REVOKE | ដកសិទ្ធិ | Remove permissions from a user or role |
| SQL Injection | ការបញ្ចូល SQL គ្រោះថ្នាក់ | Attack that inserts malicious SQL code |
| Encryption | ការអ៊ិនគ្រីប | Converting data to unreadable format |
| Audit Log | កំណត់ត្រាសវនកម្ម | Record of all database actions |
| Parameterized Query | សំណួរមានប៉ារ៉ាម៉ែត្រ | Query using placeholders instead of concatenation |
| Principle of Least Privilege | គោលការណ៍សិទ្ធិតិចបំផុត | Give only minimum permissions needed |
