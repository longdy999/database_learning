# Day 131: Data Cleaning Techniques

## Topic
Data Cleaning Techniques — Finding and fixing duplicates, NULL values, and inconsistent formatting in databases.

## English Explanation
Data cleaning is the process of detecting and correcting corrupt, inaccurate, or irrelevant records in a dataset. Real-world databases always contain messy data: duplicate rows, missing values (NULLs), inconsistent capitalization, extra whitespace, and invalid formats. Before any analysis or reporting, you must clean the data. SQL provides built-in functions like TRIM(), LOWER(), UPPER(), REPLACE(), and COALESCE() to handle these problems efficiently. A disciplined approach to data cleaning ensures reliable reports and trustworthy analytics.

## SQL Example

```sql
-- Finding duplicate records
SELECT email, COUNT(*) as duplicate_count
FROM customers
GROUP BY email
HAVING COUNT(*) > 1;

-- Removing extra whitespace with TRIM
SELECT TRIM(first_name) AS clean_name
FROM customers;

-- Standardizing text case with LOWER/UPPER
UPDATE customers
SET email = LOWER(TRIM(email));

-- Replacing unwanted characters
SELECT REPLACE(phone, '-', '') AS clean_phone
FROM customers;

-- Handling NULLs with COALESCE
SELECT
    first_name,
    COALESCE(middle_name, '') AS middle_name,
    last_name
FROM customers;

-- Finding NULL values
SELECT * FROM customers
WHERE phone IS NULL OR email IS NULL;
```

## Hands-on Practice
Create a messy dataset and clean it with these 10 common problems:


```sql
-- Create messy test data
CREATE TABLE messy_customers (
    id SERIAL PRIMARY KEY,
    name VARCHAR(100),
    email VARCHAR(100),
    phone VARCHAR(20),
    city VARCHAR(50)
);

INSERT INTO messy_customers (name, email, phone, city) VALUES
('  John Smith  ', 'JOHN@email.com', '012-345-678', 'phnom penh'),
('john smith', 'john@email.com', '012345678', 'Phnom Penh'),
('Jane Doe', NULL, '098-765-432', 'SIEM REAP'),
('Bob Wilson', 'bob@test.com', NULL, 'siem reap'),
('  Alice Brown ', 'ALICE@TEST.COM', '011-222-333', 'Battambang');

-- Fix 1: Remove leading/trailing spaces
UPDATE messy_customers SET name = TRIM(name);
-- Fix 2: Standardize email to lowercase
UPDATE messy_customers SET email = LOWER(email);
-- Fix 3: Remove dashes from phone numbers
UPDATE messy_customers SET phone = REPLACE(phone, '-', '');
-- Fix 4: Capitalize city names properly
UPDATE messy_customers SET city = INITCAP(LOWER(city));
-- Fix 5: Replace NULL emails with placeholder
UPDATE messy_customers SET email = COALESCE(email, 'no-email@unknown.com');
```
