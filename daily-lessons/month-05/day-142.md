# Day 142: Reporting Queries - Business Intelligence Basics

## Topic
Writing SQL queries for business reports: summaries, trends, comparisons, date functions, and KPI calculations.

## English Explanation
Business Intelligence (BI) turns raw database data into useful insights for decision-makers. Reporting queries summarize large datasets into meaningful numbers: total revenue, customer growth rate, average order value. Key techniques include DATE_TRUNC for grouping by time periods, EXTRACT for pulling parts of dates, date ranges for filtering, and aggregate functions for KPI calculations. Good reports answer: "How much?", "How many?", "What changed?", and "What's trending?"

## SQL Example
```sql
-- Monthly revenue report using DATE_TRUNC
SELECT
    DATE_TRUNC('month', order_date) AS report_month,
    COUNT(*) AS total_orders,
    SUM(total_amount) AS monthly_revenue,
    AVG(total_amount) AS avg_order_value,
    COUNT(DISTINCT customer_id) AS unique_customers
FROM orders
WHERE order_date >= '2024-01-01' AND order_date < '2025-01-01'
GROUP BY DATE_TRUNC('month', order_date)
ORDER BY report_month;

-- Year-over-year comparison
SELECT
    EXTRACT(MONTH FROM order_date) AS month_num,
    SUM(CASE WHEN EXTRACT(YEAR FROM order_date) = 2024 THEN total_amount ELSE 0 END) AS revenue_2024,
    SUM(CASE WHEN EXTRACT(YEAR FROM order_date) = 2023 THEN total_amount ELSE 0 END) AS revenue_2023,
    ROUND(
        (SUM(CASE WHEN EXTRACT(YEAR FROM order_date) = 2024 THEN total_amount ELSE 0 END) -
         SUM(CASE WHEN EXTRACT(YEAR FROM order_date) = 2023 THEN total_amount ELSE 0 END)) /
        NULLIF(SUM(CASE WHEN EXTRACT(YEAR FROM order_date) = 2023 THEN total_amount ELSE 0 END), 0) * 100
    , 2) AS growth_percent
FROM orders
WHERE order_date >= '2023-01-01'
GROUP BY EXTRACT(MONTH FROM order_date)
ORDER BY month_num;

-- KPI: Customer Retention Rate
SELECT
    DATE_TRUNC('month', first_order) AS cohort_month,
    COUNT(DISTINCT customer_id) AS cohort_size,
    COUNT(DISTINCT CASE WHEN months_active >= 3 THEN customer_id END) AS retained_3m
FROM (
    SELECT customer_id,
           MIN(order_date) AS first_order,
           COUNT(DISTINCT DATE_TRUNC('month', order_date)) AS months_active
    FROM orders GROUP BY customer_id
) cohorts
GROUP BY DATE_TRUNC('month', first_order);
```

## Technical Vocabulary

| English Term | Khmer Translation | Definition |
|---|---|---|
| KPI | សូចនាករសមិទ្ធផលសំខាន់ | Key Performance Indicator - measurable business metric |
| DATE_TRUNC | កាត់កាលបរិច្ឆេទ | Truncate timestamp to specified precision |
| EXTRACT | ដកចេញ | Pull specific part from a date (year, month, day) |
| Year-over-Year | ឆ្នាំធៀបឆ្នាំ | Comparing same period across different years |
| Revenue | ចំណូល | Total money earned from sales |
| Retention Rate | អត្រារក្សាអតិថិជន | Percentage of customers who return |
| Cohort | ក្រុមអតិថិជន | Group of customers sharing a characteristic |
| Trend | និន្នាការ | Direction data moves over time |
