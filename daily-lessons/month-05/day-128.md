# Day 128: Database Backup - pg_dump and mysqldump

## Topic
Why backups matter, pg_dump and mysqldump syntax, and backup types.

## English Explanation
Database backups protect against data loss from hardware failure, human error, corruption, or security breaches. A backup is a copy of your database that can be restored later. PostgreSQL uses `pg_dump` and MySQL uses `mysqldump` to create backup files. There are three main types: full backup (everything), schema-only (structure without data), and data-only (data without structure). Regular backups are essential for any production database.

## SQL Example

```sql
-- PostgreSQL: Full database backup
-- pg_dump -U username -d database_name > backup.sql

-- PostgreSQL: Schema-only backup (structure only)
-- pg_dump -U username -d database_name --schema-only > schema_backup.sql

-- PostgreSQL: Data-only backup
-- pg_dump -U username -d database_name --data-only > data_backup.sql

-- PostgreSQL: Compressed backup
-- pg_dump -U username -d database_name -Fc > backup.dump

-- PostgreSQL: Backup specific table
-- pg_dump -U username -d database_name -t table_name > table_backup.sql

-- MySQL: Full database backup
-- mysqldump -u username -p database_name > backup.sql

-- MySQL: Schema-only backup
-- mysqldump -u username -p --no-data database_name > schema_backup.sql

-- MySQL: Data-only backup
-- mysqldump -u username -p --no-create-info database_name > data_backup.sql

-- MySQL: Backup specific table
-- mysqldump -u username -p database_name table_name > table_backup.sql
```

## Speaking Practice
"Regular database backups are critical for any production system. I recommend using pg_dump for PostgreSQL databases. We can do full backups nightly and schema-only backups for version control. This ensures we can recover from any data loss situation."

## Reading Practice
Read the output of a pg_dump backup file:
- Identify the CREATE TABLE statements
- Find the INSERT statements for data
- Notice the SET statements at the beginning
- Understand the order of operations in the backup
