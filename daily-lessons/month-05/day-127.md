## English Explanation
This week we review everything learned about PostgreSQL and MySQL fundamentals. PostgreSQL is a powerful open-source relational database with advanced features like JSONB, arrays, and strong SQL compliance. MySQL is widely used in web applications and is known for speed and simplicity. Understanding both helps you work in diverse environments. Key areas: installation, connection, data types, basic commands, and the differences between the two systems.

## SQL Example

```sql
-- PostgreSQL: Check version
SELECT version();

-- PostgreSQL: List databases
-- \l

-- PostgreSQL: List tables in current database
-- \dt

-- PostgreSQL: Describe a table
-- \d table_name

-- MySQL: Check version
SELECT VERSION();

-- MySQL: List databases
SHOW DATABASES;

-- MySQL: List tables
SHOW TABLES;

-- MySQL: Describe a table
DESCRIBE table_name;

-- PostgreSQL specific data types
CREATE TABLE pg_example (
    id SERIAL PRIMARY KEY,
    data JSONB,
    tags TEXT[],
    created_at TIMESTAMPTZ DEFAULT NOW()
);

-- MySQL equivalent
CREATE TABLE mysql_example (
    id INT AUTO_INCREMENT PRIMARY KEY,
    data JSON,
    tags VARCHAR(255),
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
```
