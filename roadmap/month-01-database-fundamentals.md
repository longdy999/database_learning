## Common Mistakes to Avoid

1. **Forgetting the semicolon** - Every SQL statement ends with `;`
2. **Using spaces in table names** - Use underscores: `student_grades` not `student grades`
3. **No primary key** - Every table should have a unique identifier
4. **DELETE without WHERE** - This deletes ALL rows! Always use WHERE
5. **UPDATE without WHERE** - This updates ALL rows! Always use WHERE
6. **Wrong data types** - Don't store numbers as text
7. **Inconsistent naming** - Pick lowercase_with_underscores and stick with it
