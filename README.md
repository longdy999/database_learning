# Database Learning - 180 Days Professional Program

## From Beginner to Professional Database Skills in 6 Months

**Duration:** 180 days | **Daily commitment:** 30 minutes | **Level:** Beginner to Professional

---

## What This Repository Contains

This is a complete, structured database learning program designed to take you from zero database knowledge to professional-level skills in 6 months.

### Program Goals
1. Pass a database/SQL certification exam
2. Understand databases from beginner to professional level
3. Explain database concepts clearly to clients
4. Join daily technical conversations about databases confidently
5. Write clean, professional SQL queries
6. Design practical real-world databases
7. Understand how databases are used in software, business, reporting, and backend systems

## 6-Month Roadmap Overview

| Month | Theme | Key Skills |
|-------|-------|-----------|
| 1 | Database Fundamentals | Tables, records, columns, keys, basic SQL |
| 2 | SQL Querying | Filtering, joins, grouping, aggregation |
| 3 | Database Design | Normalization, ERD, relationships, constraints |
| 4 | Advanced SQL & Performance | Subqueries, views, indexes, transactions |
| 5 | Real-World Database Work | PostgreSQL, backups, security, APIs, reporting |
| 6 | Professional Readiness | Exam prep, portfolio projects, client communication |

## How to Use This Repository

### Daily Routine (30 minutes)
1. **Open today's lesson** in `daily-lessons/month-XX/day-XXX.md`
2. **Read the topic** (5 min) - English + Khmer explanation
3. **Study the SQL example** (5 min)
4. **Hands-on practice** (10-15 min)
5. **Vocabulary + Speaking practice** (5 min)
6. **Review task** (5 min)

### Weekly Routine
- Complete the weekly test in `weekly-tests/week-XX.md`
- Do the speaking challenge
- Review your mistakes

### Monthly Routine
- Complete the mini-project in `projects/month-XX-mini-project/`
- Review the month's vocabulary
- Update your progress tracker

## Tools You Need
- **SQLite** (built into most systems) - for Month 1-3
- **PostgreSQL** or **MySQL** - starting Month 4-5
- **DB Browser for SQLite** (free GUI tool) - optional but helpful
- **Any text editor** - VS Code recommended
- **Terminal/Command Line** - for running SQL
